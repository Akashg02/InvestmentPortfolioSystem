GET/api/clients  List all clients for authenticated advisor 🔒 JWT
POST/api/clients  Create new client 🔒 JWT
GET/api/portfolios/{id}  Get portfolio with full investment details 🔒 JWT
POST/api/portfolios/{id}/refresh-prices  Fetch live market prices for all holdings 🔒 JWT
POST/api/investments  Buy an investment (merges if symbol exists) 🔒 JWT
POST/api/investments/{id}/sell  Sell shares — partial or full position 🔒 JWT
GET/api/market-data/quote/{symbol}  Real-time stock quote via Alpha Vantage 🔒 JWT
GET/api/market-data/summary  S&P 500, NASDAQ, DOW summary 🔒 JWT
GET/health  Application health check endpoint
